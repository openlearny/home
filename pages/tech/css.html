What is CSS?

CSS (Cascading Style Sheets) controls how HTML elements look on the screen. It handles layout, colors, spacing, fonts, and responsiveness. 

h1 {
    color: red;
    font-size: 32px;
}

How CSS Works

CSS follows three core rules: cascade, specificity, and inheritance. When multiple styles apply, the browser decides which one wins. 

p {
    color: blue;
}

.text {
    color: green;
}

CSS Syntax

CSS is written as selector + property + value. 

selector {
    property: value;
}

Selectors

Selectors define which HTML elements the styles apply to. 

/* Element */
p {}

/* Class */
.card {}

/* ID */
#header {}

Box Model

Every element is a rectangular box consisting of content, padding, border, and margin. Misunderstanding this causes 80% of layout bugs. 

.box {
    width: 200px;
    padding: 16px;
    border: 2px solid black;
    margin: 12px;
}

Colors & Units

CSS supports multiple color formats and measurement units. Avoid magic numbers; use relative units. 

body {
    color: #333;
    font-size: 1rem;
    width: 80%;
}

Specificity

Specificity decides which rule overrides another. Inline styles > IDs > Classes > Elements. 

#title { color: red; }
.title { color: blue; }
h1 { color: green; }

Display

The display property controls how elements behave in the layout. 

span {
    display: inline-block;
}

Flexbox

Flexbox is a one-dimensional layout system designed for alignment. Use it for navbars, cards, and centering. 

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

Grid

CSS Grid is a two-dimensional layout system for complex layouts. Use it for full page structures. 

.layout {
    display: grid;
    grid-template-columns: 1fr 3fr;
    gap: 16px;
}
